About

A simple web server built with Node.js and Express. A personal project for learning the basics of backend development.

What it does

Serves static pages, handles API routes, and returns JSON responses — the building blocks of any web server.

Why I built it

To get hands-on experience with Express routing, middleware, and serving files over HTTP.
